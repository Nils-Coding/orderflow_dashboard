# Orderflow Dashboard

Dieses Repository enthält die Frontend‑App unter `orderflow-frontend/` (Astro + Vue + Tailwind).

## Schnellstart

```bash
cd orderflow-frontend
npm install
cp env.example .env
npm run dev
```

Öffne anschließend die URL aus dem Terminal (normalerweise `http://localhost:4321`).
